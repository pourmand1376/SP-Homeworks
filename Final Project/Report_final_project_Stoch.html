Stationary Distribution

Suppose that we have a markov chain model. The transition matrix (or transition kernel) is known. Therefore, we can easily get stationary distribution, i.e., the probabiliy of staying in each state in the long run.

For example, suppose that we have a transition matrix PPPP and we want to understand stationary distribution of PPPP:

P = \begin{bmatrix}
0.1 & 0.2 & 0.7\\
0.3 & 0.6 & 0.1\\

def get_stationary_distribution(Q):
    evals, evecs = np.linalg.eig(Q.T)
    evec1 = evecs[:,np.isclose(evals, 1)]
    evec1 = evec1[:,0]

Metropolis-Hastings Algorithm

Now, suppose that we want to sample from this markov chain. There are two algorithms. One is called metropolis which works only on symmetric transition matrices in which p_{ij}=p_{ji}pij=pjip_{ij}=p_{ji}pij​=pji​. But this is not always possible. Hasting, hence the name, extended the algorithm to be used with non-symmetric matrices.

We start from random state xxxx. Then we choose one of its neighbours based on the transition probabiliy. We either accept it or reject it and stay where we are. This is the gist of algorithm.

First you need to start from a random state, let's call it X_0X0X_0X0​.

1- Sample a candidate point YYYY from our transition matrix given previous XXXX.
2- Compute the ration r=\frac{\pi(y) p(y,x) }{ \pi(x)p(x,y)}r=π(y)p(y,x)π(x)p(x,y)r=\frac{\pi(y) p(y,x) }{ \pi(x)p(x,y)}r=π(x)p(x,y)π(y)p(y,x)​
3- Compute \alpha = min(r,1)α=min(r,1)\alpha = min(r,1)α=min(r,1)
4- Accept YYYY with probabiliy \alphaα\alphaα or reject it with probabiliy 1-\alpha1−α1-\alpha1−α

Do this process iteratively and you will have a sample from markov chain.

def metropolis_hastings(sample_items, transition, stationary_dist):
    sample = np.zeros(sample_items,dtype=np.int32)
    for i in range(sample_items-1):
            x = sample[i]
            y = w_choice(transition[x])

            r_nominator = stationary_dist[y]*transition[y,x] 
            r_denominator = stationary_dist[x]*transition[x,y]

            a = 0
            if r_denominator == 0:
                a = 1
            else:
                a = min(r_nominator / r_denominator,1)

            if random.uniform(0,1) <= a:
                sample[i+1] = y
            else:
                sample[i+1] = x
    return sample

from random import uniform
import random
def w_choice(seq):
    """
    choose an item from list of items whose probability is given
    example: input [0.3,0.4,0.3] 
    output: it may choose from [0,1,2] with given probabilty
    """
    total_prob = sum([item for item in seq])
    chosen = random.uniform(0,total_prob)
    cumulative = 0 
    for i in range(len(seq)):
        cumulative += seq[i]
        if cumulative > chosen:
            return i
